# 项目路线图

## 阶段 1：基础实验台搭建
- 建立二维平面环境
- 设置起点、终点、障碍物
- 可视化机器人运动轨迹
- 实现简单运动学模型

## 阶段 2：复现 baseline MPPI
- 实现控制序列采样
- 轨迹 rollout
- 代价计算
- 加权更新控制序列
- 完成基本避障导航

## 阶段 3：设计改进算法
- 改进采样策略
- 尝试 guided sampling
- 分析不同参数对性能的影响

## 阶段 4：实验与对比
- 稀疏障碍场景
- 密集障碍场景
- 窄通道场景
- 统计成功率、轨迹长度、规划时间等指标

## 阶段 5：阶段总结
- 撰写实验报告
- 形成汇报材料
- 准备后续向实机迁移
