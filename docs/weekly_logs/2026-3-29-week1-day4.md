2. 它的核心流程可以概括为：  
   **采样控制/噪声 → rollout 轨迹 → 计算 cost → 计算权重 → 加权更新控制序列 → 执行第一步 → 滚动重规划**。

   **把 MPPI 和 learned dynamics / neural network dynamics 结合起来，并且希望摆脱对 control-affine dynamics 的严格依赖。**
